### 더 익숙해지면 좋을 VSC 단축키

alt + f12 → 해당 함수가 어떻게 구성됐는지 새 창으로 확인 가능

f2 → rename (다른 파일에 참고가 걸려 있는 변수명, 함수명 등도 같이 바꿔주기에 ctrl + h보다 좀 더 스마트하고 안전함)

ctrl + L → 한 줄 code highlight

ctrl + 좌우방향키 → 이동 시 좀 편하게 워프

ctrl + p → 파일간 이동 단축키로 가능

ctrl + ` → 터미널창 열기
