### VSC 단축키 모음

- ctrl + / → 주석
- ctrl + b → 사이드바 토글
- ctrl + y → 되돌리기한 거 다시 복원(ctrl + shift + z와 동일)
- ctrl + shift + f → 전체에서 찾기
- ctrl + g → 행 숫자 입력으로 인한 빠른 행 이동
- ctrl + d → 일치 단어 선택
- alt + 좌클릭 → 멀티 커서
- ctrl + home or end → 페이지 최상단/최하단 이동
- ctrl + pageup or pagedown → 열려 있는 탭 이동
- ctrl + enter → 커서 위치와 상관없이 바로 다음 행 이동
- ctrl + ` → 터미널 토글
